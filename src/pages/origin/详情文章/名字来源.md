# 背景
在还没有接触电脑之前我很少考虑**命名**这件事，那时的我还很稚嫩，能接触到的东西也有限，名字对我来说不过是个临时的标识
但是在高考之后的那个暑假一切都变了。那个暑假，我拥有了人生中第一台真正属于我的电脑。也正是从那时开始，我第一次频繁地面对各种“命名”问题

可怜愚蠢的我第一个用户名写的是我自己的真实姓名。我记得当时是看注册的时候提示是姓名，我心想姓名这么正式，不像昵称什么的，再加上写上自己的名字也好知道是谁的，所以就写自己的名字了。
后来，我玩了一段时间电脑后就遇到了路径问题，可惜当时我只知道某些路径不能包含中文，不知道有些路径还不能含空格。也是在处理这些问题的过程中，我开始意识到：一个名字不只是看起来好听，它还要能适应不同平台、不同系统和不同场景

于是，我开始认真考虑一个问题：我是否应该拥有一个相对固定、同时又真正属于自己的名字

# 介绍
**Dead soul insect**：死魂虫，原名：しにだまちゅう
出自[《犬夜叉》](/origin/详情文章/犬夜叉/)，是一种妖怪，主要作用是可以收集死者的魂魄供缺乏灵魂的躯体继续活动。由于死魂虫会飞，所以也可以骑在它身上飞，或者是让它们缠着人的身子飞行
在《犬夜叉》里，死魂虫主要服务于桔梗。它们负责为桔梗那具由泥土和骨灰制成的躯体补充死魂，使其具备行动能力。死魂虫在《犬夜叉》里不占据太多笔墨，基本上起个暗示作用：当死魂虫出现时，往往意味着桔梗就在附近，或者即将登场

# 为什么起这个名字
《犬夜叉》对我颇具影响，这是我在很稚嫩的时候看的第一部动漫，我对其中出场的众多角色都很喜欢甚至是欣赏。
所以当我要重新命名的时候，我就自然而然地想到了《犬夜叉》。不过，如果直接以其中某个角色命名对于我来说太过肤浅，我并不想顶替着他们的名号做事，而且大多角色的中文名没有足够的意境，或者具备某种抽象的代表性。甚至有些角色的名字偏显中二，所以这个方案很快就被我排除了
接下来，武器名？稍微想下也感觉不太合适，况且也没有好听的名字

思来想去想到了桔梗身边总是会出现的**死魂虫**
这个好：哪怕是中文名，对于不了解的人来说也算正常：末尾是“虫”，可能会想到是什么虫子？“死魂“也不算太中二，甚至在如今的很多作品中，亡灵类元素还是挺常见的
而且“死魂虫”这个名字可以轻松扩展成各种衍生称呼，比如直接大、小死魂虫，或者来个形容词描述下什么样的死魂虫。这样就可以轻松应对部分平台不可重复取名的问题
最终，我就叫**死魂虫**了

因为前车之鉴，所以我的用户文件夹要改成英文名：Dead soul insect。接下来还有个问题：用复数还是单数？
用单数是表示个体，更像是说“我是某只死魂虫”；而用复数则更像是在表达一种集合性的、抽象的概念。相比之下，复数形式更符合我想要的感觉
最终就得到了Dead soul insects这个名字
