# 背景
在还没有接触电脑的时候我没怎么考虑**命名**这件事，当时我还很稚嫩，接触不了太多。但是在高考之后的那个暑假一切都变了。
因为我对电脑一直很憧憬：有不少想玩的电脑游戏而且总感觉电脑比手机好尽管当时还什么都不懂……再加上我对大事物的第一次都比较重视，所以我对电脑可谓是精挑细选，不惜看几个小时视频去了解哪台电脑好，可惜最终我还是没能忍住诱惑，买了个当时的热款，好在还是不亏的，我对这台电脑很满意。可怜愚蠢的我第一个用户名写的是我自己的名字。。。我记得当时是看注册的时候提示是姓名，我一看姓名这么正式，不像昵称什么的，再加上写上自己的名字也好知道是谁的，所以就写自己的名字了。
后来玩了一段时间的电脑后我就遇到了路径问题，可惜当时只知道有些路径不能包含中文，不知道有些还要求不能含空格。所以我就把用户文件夹改成：**Dead soul insects**。以我现在的眼光看，应该改成**dead-soul-insects**的

# 介绍
**Dead soul insect**：死魂虫。原名：しにだまちゅう
出自《犬夜叉》，是一种妖怪，主要作用是可以收集死者的魂魄供缺乏灵魂的躯体活动，当然，由于死魂虫会飞，所以也可以骑它身上来飞，或者是让死魂虫缠着人的身子飞行
